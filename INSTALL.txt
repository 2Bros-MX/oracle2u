ORACLE2U Extension - Installation Guide
========================================

QUICK INSTALL (Windows):
1. Double-click "install.bat"
2. Wait for installation to complete
3. Follow the on-screen instructions

MANUAL INSTALL:
1. Make sure Node.js is installed (download from https://nodejs.org/)
2. Open Command Prompt in this folder
3. Run: npm install
4. Run: npm run build
5. Open Chrome and go to chrome://extensions/
6. Enable "Developer mode"
7. Click "Load unpacked"
8. Select the "dist" folder

TROUBLESHOOTING:
- If install.bat doesn't work, try running it as Administrator
- Make sure Node.js is installed and added to your PATH
- Check that you have internet connection for npm install
